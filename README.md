# Dom-Implementation
My DOM practice project – interactive color boxes and a personalized greeting using JavaScript.
# 🎨 Color Greeting App

## About this project

This project was built as part of my DOM assignment.
The goal was to understand how JavaScript can interact with HTML elements and make a webpage dynamic.

Instead of keeping it basic, I tried to make the UI a bit cleaner and more interactive.

---

## What it does

* Takes user input (name) and displays a personalized greeting
* Shows 4 boxes (Red, Blue, Green, Yellow)
* Clicking on a box fills it with that color
* Adds small hover effects for better user experience

---

## Why I built this

I wanted to get comfortable with:

* DOM manipulation
* Handling user input
* Writing cleaner JavaScript (instead of inline code)

This is one of my early frontend projects, so the focus was more on learning than complexity.

---

## Tech used

* HTML
* CSS
* JavaScript

Just open the `index.html` file in your browser.
